# 💖 Love Calculator (Python CLI Project)

A fun beginner Python project that generates a random "love score" between two names and displays a playful compatibility message based on the score.

This project is built for learning Python basics like functions, loops, conditionals, input handling, and simple validation.

---

## 🚀 Features

- Takes user names as input
- Validates name input (only alphabetic characters allowed)
- Generates a random love score (0–100)
- Displays different messages based on score range
- Allows repeated usage until the user exits
- Beginner-friendly CLI interface

---

## 🧠 How It Works

1. User enters their name and their partner’s name
2. Program validates input
3. User presses Enter to generate a score
4. A random number between 0 and 100 is generated
5. A message is displayed based on the score
6. User chooses to continue or exit
